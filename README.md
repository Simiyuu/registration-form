# Responsive Registration Form

A clean, accessible, and fully responsive registration form built with HTML5 and CSS3. This project demonstrates modern frontend development practices, including semantic markup, client-side form validation, and responsive design techniques.

## 📋 Features

* **Responsive Layout:** Utilizes CSS viewport units (`vw`) and `min/max-width` properties to ensure the form scales perfectly across mobile, tablet, and desktop screens.
* **Client-Side Validation:** Implements native HTML5 validation including `required` attributes, `min`/`max` value constraints for numbers, and RegEx `pattern` matching for secure password creation.
* **Accessibility (a11y) Focused:** Uses semantic tags (`<fieldset>`, `<legend>`) and explicitly linked `<label>` elements to ensure compatibility with screen readers and improve overall user experience.
* **Custom Dark Theme:** Features a cohesive, dark-themed UI (`#1b1b32` background) with styled inputs that override default browser aesthetics for a polished, corporate look.
* **Advanced CSS Selectors:** Employs pseudo-classes (`:last-of-type`) and attribute selectors (`input[type="submit"]`) to maintain clean, efficient, and modular stylesheets.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure and form controls.
* **CSS3:** Custom styling, box-model spacing, and responsive flex/block behavior.

## 🚀 Getting Started

To view this project locally, simply clone the repository and open the HTML file in any modern browser.

### Prerequisites
No external libraries or build tools are required. 

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/yourusername/registration-form.git](https://github.com/Simiyuu)

```

2. Navigate to the project directory:
```bash
cd registration-form

```


3. Open `index.html` in your web browser.

* `index.html` - Contains the semantic structure and form elements.
* `styles.css` - Contains the responsive layout and dark-theme styling rules.
